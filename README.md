# IronSleep_data_processing
Repository containing the code of the data processing of the IronSleep data, containing MPM processing with the hMRI toolbox, T2 data generation from multi-echo spin-echo data, and QSM calculations from phase measurements.


## Steps:

- `symlink_rename_files.py`: create symlinks that rename the LORAKS reconstructed files, to meet naming requirements of QSMxT
